About KAT

The Kmer Alignment Tool takes DNA genome sequences, choppes them up into kmers, and then aligns those kmers to Reference DNA files. For the alignment it uses a combination of Suffix Arrays and the Burrows-Wheeler Transform.

Use

If you sign is as a Guest, the result files will be uploaded to the webpage after the program is finished. Once you start a run, do not close the page or you won't be able to access your results.
If you Register, then you will have access to your user profile where all your runs will be saved. You can close the page after starting a run and the results will be uploaded to your profile once the run is finished.
You can only have one run at the time, otherwise you will get some funny results.

Input

KAT requires 4 inputs:

  • Project name: Name of the run
  • Kmer size: This should be an integer between 5 and 1000. It represents the number of basepairs for chopping up the samples.
  • Sample file: Up to 5 genome DNA sequences in fasta files. These are the samples that will be chopped into kmers.
  • Reference file: Up to 5 genome DNA sequences in fasta files for the alignment.

For the fasta files, the program can handle genomes with more than one sequence(e.g. chromosomes and contigs).

Results

A zip file can be downloaded that will include the following:

  • Alignment table: Contains the hits and hit percentages.
  • Plots of hits and hit percentages.
  • Alignment textfiles: A .txt file is created for each sample-reference pair. The file includes a list of the kmer sequences (from the sample) and their positions in the reference genome. In the example, kmer AAACTGGGCAGCAGGGCGGG is found in Reference NZ_CP055292.1 sequence 0, position 3383183.

About KAT

KAT is Kmer Alignment Tool that divides uses Suffix Arrays and the Burrows-Wheeler Transform in order to align DNA sequences. The software takes

Input

KAT requires 4 inputs:

  • Project name:Name of the run
  • Kmer size: One integer from 5 to 1000
  • Sample file:A DNA sequence in a fasta file
  • Reference file:A DNA sequence in a fasta file

For the fasta files, the program can handle genomes with more than one chromosome and contigs.

Results

sd

  • Alignment table:Contains the hits and hit percentage
  • Alignment positions:A .txt file is created for each sample-reference pair. The file includes a list of the kmer sequences (from the sample) and their positions in the reference genome.
